Semester 3 - Summmer 2015

Front End

Amazon

This project had us design a bookstore product page similar to amazon.com. This site is one page. All code is done entirely from scratch, no frameworks used. Styling the triple row header without any grid system was great practice.

Autoblox

This project involved taking content and images from an existing company and using it to style a five-page website of our own. This project was an introduction to bootstrap for me. I enjoyed playing around with the vertical accordion feature on the home page.

Human Tree Inc

For this project we had to choose a logo and then build a web page that visually correlated with it. I chose a nature-themed approach, and overall like the result of it. I feel the narrow style of the page works in this setting, and the background really adds to the vibe. This was my very first attempt with bootstrap, or any framework for that matter.

Yahoo

For this Project we had to take a popular existing website and create a three-page alteration to how we believe the site could be more friendly. Yahoo.com is a very cluttered page, and I felt it could try an approach similar to Google for their main page. Time restraints made it difficult to complete this project to my satisfaction, as there is lots more I’d like to do. No frameworks were used.

Back End

Game Calculator (Currently Unavailable)

This project is programmed with C# in ASP.NET. We had to acquire many formulas based on user input for the four games. All form data is fully validated.

Student Board (Link Currently Unavailable)

This was an in class assignment with our added alterations. We created a student record application, where user data creates student records, signifying which department, and course each student is enrolled in.

Semester 2 - Winter 2015

Front End

Projects
Magazine Layout

For this project we were handed a PDF of a real magazine advertisement and we were to recreate the page in the browser to the best of our ability. My recreation is a stunning duplicate of the original.

Ritters

For this project we were to design a web page for a candy of our choice. I am a chocolate fanatic and had Ritters on the mind, so that’s where this went. This was an exploration project, learned a lot about what works and what doesn’t work at this time.

Typefaces

This project was useful to understand the use of typefaces, and floats. I also explored background opacity. The need for responsive design became apparent to me during this assignment, we touched into that later.

Back End

Content Management System (Currently Unavailable)

This project was the largest task so far in this program. We used PHP to create a Content Management System(CMS) from scratch. We learned how to create, read, update, and delete from the database. Creating this full-featured CMS was an amazing experience to have a fully functioning template of my own to work with for employers.

Music SQL

This project involved understanding the concepts of SQL. We took a list of songs, and learned how to organize and manipulate the table data by a variety of different sorting.

Graphic Design

Lunch

This assignment involved the use of gradients in illustrator. We were given a colorless image model and had to create our own version with color.

Frog

This project was created an Illustrator. We were to explore the pen tool by creating the image based on the provided example.

Clothing Sale

Using Illustrator we had to create a clothing sale advertisement for an item of our choosing.

Yoga

For this project we used Photoshop C6 redesign the right portrait image. My result is the left portrait. Learned lots about layering, scaling, blur, color replacement, and lots more.

Semester 1 - Fall 2014

Front End

Phone Book

My first large scale CSS project. We designed a five-page website about our chosen old historical item. I heavily focused on learning the grid system, and making sure everything responded inline with one another. For our knowledge level at this point I am satisfied with the my result.

Front End

Site Creation - Photoshop

For this project we designed a full web page layout with a software of our choice, I used Photoshop. Our task was to integrate our design knowledge into a clean, friendly layout of our choosing. I chose a very busy concept, but for all I wanted to include in the site I feel I managed the space quite well
